# MDT Admin

Admin dashboard for managing dummy tickets, insurance applications, users, and blog content.

## Stack
- Vite + React
- React Router
- TanStack Query
- Tailwind CSS

## Setup
1. Install dependencies:
   - `npm install`
2. Run the dev server:
   - `npm run dev`

## Environment Variables
Create a `.env` file in `mdt-admin/`:
- `VITE_BACKEND` — backend base URL
- `VITE_TINYMCE_API_KEY` — TinyMCE API key

## Scripts
- `npm run dev` — start dev server
- `npm run build` — build for production
- `npm run preview` — preview production build
- `npm run lint` — lint
